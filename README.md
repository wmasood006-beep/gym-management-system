# Gym Management System

This is a basic Gym Management System website created using Flask, HTML, and CSS.

## Features
- Home page with welcome message
- About Us page with gym details
- Membership page with plans
- Flask routing and templates
- CSS styling

## Folder Structure
- app.py
- templates (HTML files)
- static (CSS file)

## How to Run
1. Install Flask  
   pip install flask

2. Run the app  
   python app.py

3. Open browser  
   http://127.0.0.1:5000/
